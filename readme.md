# Ayn Socket Framework 

## 这是什么 | What`s this?

通过一些原生的Python库实现的套接字（Socket）服务框架...

该项目由曾经的goSocket（组播实现）、goRC（远程控制）和goRPC（远程过程调用）组合而成，而且将会添加更多功能。顺带一提，前面的go和golang无关...
